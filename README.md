## General info
This project is built around the macOS notepad design as well as Evernote.


	
## Technologies
Project is created with:
* ASP.NET Core version: 3.1
* WPF

	
## Setup
To run this project, install it locally using npm:

```
$ cd ../lorem
$ npm install
$ npm start
```
